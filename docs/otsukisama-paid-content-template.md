# おつきさま診断 - 有料版コンテンツテンプレート

## 概要

このテンプレートは、現在の無料LP構造（lp-otsukisama.html）に基づいて、有料版で提供される完全なコンテンツを定義しています。

## 無料版 vs 有料版の違い

### 無料版で提供される内容
- フォーム入力
- 月詠の挨拶
- 6つのタイプ診断結果（円形配置）
- 基本月相と裏月相の説明
- 3つの月の力（簡潔版）
- 4つの軸（簡潔版）
- 運勢セクションの導入部分（ぼかし付きでテイスト部分のみ）

### 有料版で追加される内容
- 詳細な今後1年間の全体運勢
- 完全な恋愛運分析
- 詳細な人間関係運
- 仕事運・キャリア診断
- 金運・財運分析
- 個人専用月齢カレンダー
- 特別な月の教えとメッセージ

```javascript
// 無料版表示の制御
function showFreeContent() {
    // 基本情報は表示
    document.getElementById('basic-diagnosis').style.display = 'block';
    document.getElementById('moon-phase-basic').style.display = 'block';
    
    // 運勢セクションはテイスト版のみ表示（ぼかし付き）
    document.querySelectorAll('.fortune-section .section-header').forEach(el => {
        el.style.display = 'block';
    });
    
    // 詳細コンテンツは非表示
    document.querySelectorAll('.fortune-content').forEach(el => {
        el.style.display = 'none';
    });
    
    // 「続きを見る」ボタンを表示
    document.getElementById('paid-content-cta').style.display = 'block';
}

// 有料版コンテンツの表示
function showPaidContent() {
    // 全コンテンツを表示
    document.querySelectorAll('.fortune-content').forEach(el => {
        el.style.display = 'block';
    });
    
    // 月齢カレンダーを表示
    document.getElementById('personal-moon-calendar').style.display = 'block';
    
    // 最終メッセージを表示
    document.getElementById('final-detailed-message').style.display = 'block';
    
    // CTAボタンを非表示
    document.getElementById('paid-content-cta').style.display = 'none';
}
```

### 2. コンテンツの動的生成

各月相タイプ、裏月相タイプの組み合わせに応じて、テンプレートから適切なコンテンツを生成する仕組みが必要です。

### 3. 月齢カレンダーの動的計算

現在日時から2週間分の月齢を計算し、各日の運勢とアドバイスを動的に生成する必要があります。

### 4. 決済システムとの連携

- 決済完了後に有料コンテンツを表示
- 閲覧期間の制限（10日間）
- 再アクセス時の認証

このテンプレートは、現在のLP構造を完全に踏襲しつつ、有料版でのみ提供される詳細なコンテンツを体系的に定義しています。開発者はこのテンプレートに基づいて、月相パターンに応じた動的コンテンツ生成システムを構築できます。
