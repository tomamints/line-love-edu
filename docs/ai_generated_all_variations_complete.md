## 💕 4つの恋愛タイプの組み合わせパターン

### 感情表現タイプ（4種類）
1. **ストレート型**：素直に気持ちを伝える
2. **ほのめかし型**：遠回しに伝える
3. **行動型**：言葉より行動で示す
4. **サポート型**：相手を支えることで愛を表現

### 距離感タイプ（4種類）
1. **密着型**：常に一緒にいたい
2. **バランス型**：適度な距離を保つ
3. **自由マイペース型**：お互いの時間を大切に
4. **尊重型**：相手のペースに合わせる

### 価値観タイプ（4種類）
1. **ロマンチスト型**：運命の恋を信じる
2. **現実主義型**：安定と信頼を重視
3. **冒険型**：刺激的な恋愛を求める
4. **調和型**：穏やかな関係を築く

### エネルギータイプ（4種類）
1. **燃え上がり型**：情熱的に燃える
2. **持続型**：一定の温度を保つ
3. **波型**：感情の起伏がある
4. **穏やか型**：静かに愛を育む
