## 🎯 決定ロジック完全版

### STEP1: ユーザー情報の取得
```
入力情報：
- 生年月日（必須）
- 性格アンケート結果（任意）
  - エネルギータイプ
  - 価値観タイプ
  - 距離感タイプ
  - 感情表現タイプ
```

### STEP2: 月相タイプの判定
```
計算方法：
1. 生年月日から当日の月齢を算出（0〜29.5）
2. 月齢を8分割して月相タイプを決定

月齢範囲 → 月相タイプ：
0〜3.7 → 新月
3.7〜7.4 → 三日月
7.4〜11.1 → 上弦の月
11.1〜14.8 → 十三夜
14.8〜18.5 → 満月
18.5〜22.2 → 十六夜
22.2〜25.9 → 下弦の月
25.9〜29.5 → 暁
```

### STEP3: 裏月相の判定
```
計算式：
(月相番号 + 誕生月 + 誕生日の数字和) % 8

例：1995年3月15日、月相が満月（4）の場合
→ (4 + 3 + 6) % 8 = 5（十六夜）
```

### STEP4: 運勢パターンの選択
```
パターン決定：
月相タイプ × 裏月相タイプ = 64パターンから選択

性格アンケートがある場合：
恋愛運の表現を性格タイプに合わせて調整
```
