# 年間運勢システム設計書

## 概要
おつきさま診断における年間を通じた長期的運勢システム。日単位の変動ではなく、月相パターンに基づいた一年間の大きな流れと季節的なエネルギー変化を重視したアプローチ。

---

## 基本コンセプト

### 従来システムからの変更点
1. **日単位変動の廃止**: 毎日変わる運勢スコアを廃止
2. **年間テーマの導入**: その人の一年間を貫く基本テーマを設定
3. **季節的フロー**: 自然のリズムに合わせた四季の運勢変化
4. **転換点の明確化**: 年間を通じた重要な変化のタイミング
5. **質的表現**: 数値化ではなく、物語性のある質的な表現

---

## 年間運勢の構造

### 1. 年間基本テーマ（Annual Core Theme）
64パターン（月相×裏月相）ごとに設定される、その人の一年を貫く基本的な流れ

#### テーマ例：
- **新月×新月**: 「創造と始まりの年」
- **満月×三日月**: 「完成から成長への循環」
- **下弦の月×上弦の月**: 「効率と決断の調和」
- **暁×暁**: 「変革と適応の年」

### 2. 季節的エネルギーフロー（Seasonal Energy Flow）
一年を通じた自然なエネルギーの変化と、各月相タイプへの影響

#### 春（3-5月）: 芽吹きと始動
- **新月系タイプ**: 最高エネルギー期、新規プロジェクト開始に最適
- **三日月系タイプ**: 成長エネルギー活性化、スキルアップに最適
- **上弦の月系タイプ**: 戦略立案期、長期計画の策定に適している
- **十三夜系タイプ**: 安定的成長期、基盤強化に向く
- **満月系タイプ**: 情熱上昇期、大きな目標設定に適している
- **十六夜系タイプ**: 成熟した判断期、経験を活かした展開
- **下弦の月系タイプ**: 効率化推進期、システム改善に最適
- **暁系タイプ**: 変化対応期、新しい環境への適応

#### 夏（6-8月）: 活動と実現
- **全タイプ共通**: 活動エネルギーが最高潮に達する期間
- **恋愛運**: 全般的に恋愛運が高まり、情熱的な展開が期待
- **仕事運**: 成果を出すのに最適な時期、大きなプロジェクトの完成
- **対人運**: 新しい出会いや既存関係の深化

#### 秋（9-11月）: 収穫と調和
- **満月系タイプ**: 完成エネルギー最高潮、大きな成果を収穫
- **十三夜系タイプ**: 安定と調和の完成形、理想的な状態の実現
- **下弦の月系タイプ**: 効率の成果が見える化、システムの完成
- **全体的**: 春に始めたことの成果が現れる時期

#### 冬（12-2月）: 内省と準備
- **暁系タイプ**: 内面充実期、次の変化への準備に最適
- **三日月系タイプ**: 自己成長期、スキルや知識の蓄積
- **全体的**: 外向きよりも内向きのエネルギーが強い時期
- **来年準備**: 次年度の計画と準備に適した期間

### 3. 年間転換点（Annual Turning Points）
一年を通じて訪れる重要なエネルギーシフトのタイミング

#### 主要転換点：
1. **春分～5月初旬**: 「始動の転換点」
   - 新しいサイクルの開始
   - 重要な決断と行動開始の時期
   
2. **夏至前後（6月）**: 「活動の頂点」
   - エネルギーが最高潮に達する
   - 大きな成果や成就の時期
   
3. **秋分～10月**: 「収穫と統合」
   - 上半期の成果を確認・統合
   - 関係性の深化や安定化
   
4. **冬至～年末**: 「内省と準備」
   - 一年の振り返りと来年への準備
   - 内面的な成長と充実

## 運勢カテゴリー別の年間パターン

### 全体運の年間フロー
月相パターンごとの一年間を通じた全体的な運気の流れ

#### 新月系パターンの年間フロー例：
```
1-2月: 準備期間（次の始まりに向けたエネルギー蓄積）
3-5月: 開始期（新プロジェクト・新生活のスタート）
6-8月: 発展期（始めたことが軌道に乗り、大きく展開）
9-11月: 成果期（春の種まきが実を結ぶ時期）
12月: 統合期（一年の成果を統合し、次への準備）
```

#### 満月系パターンの年間フロー例：
```
1-3月: エネルギー充電期（内面の豊かさを蓄える）
4-6月: 開花期（才能や魅力が最大限に発揮される）
7-9月: 完成期（大きなプロジェクトや関係が完成）
10-12月: 統合・指導期（成果を活かして他者をサポート）
```

### 恋愛運の年間パターン
64パターン×個性要素の組み合わせによる、個人化された恋愛運の年間フロー

#### 基本構造：
1. **出会いの時期**: いつ、どのような出会いが期待できるか
2. **関係発展期**: 関係が深まる時期とそのパターン
3. **試練・調整期**: 関係の課題と成長の機会
4. **安定・深化期**: 関係が安定し、さらに深まる時期

#### 個性要素の統合：
- **エネルギータイプ**: 恋愛エネルギーの年間変化パターン
- **価値観タイプ**: 重視する価値観の季節的変化
- **距離感タイプ**: 適切な距離感の年間調整
- **感情表現タイプ**: 表現スタイルの季節的適応

### 仕事運の年間パターン
キャリア発展の大きな流れと、各時期に適した行動指針

#### 月相タイプ別の仕事運年間パターン：

**革新系（新月、新月×暁など）**:
- 春: 新規プロジェクト立ち上げ
- 夏: アイデアの実現と拡大
- 秋: 成果の確認と次の準備
- 冬: スキルアップと新技術習得

**成長系（三日月系パターン）**:
- 春: 能力向上と資格取得
- 夏: 実績の積み重ね
- 秋: 昇進・昇格のチャンス
- 冬: 専門性の深化

**管理系（上弦の月系パターン）**:
- 春: チーム編成と戦略立案
- 夏: プロジェクト管理と成果創出
- 秋: 評価と次期計画
- 冬: リーダーシップスキル向上

### 金運の年間パターン
資産形成と金銭管理の年間戦略

#### 投資・資産運用の年間サイクル：
1. **計画期（1-3月）**: 年間の資産目標設定と戦略立案
2. **実行期（4-8月）**: 投資の実行と活発な資産運用
3. **収穫期（9-11月）**: 利益確定と資産の見直し
4. **整理期（12月）**: 税務対策と来年への準備

#### 月相パターン別の金運傾向：
- **新月系**: 新しい投資先開拓、ベンチャー投資
- **成長系**: 長期的な資産形成、定期的な積立
- **安定系**: 堅実な投資、リスク管理重視
- **完成系**: 大きなリターン獲得、投資成果の実現

## 年間一貫性の保証システム

### 1. テーマ一貫性管理
年間を通じて基本テーマがぶれないよう、各時期の内容を調整

### 2. 季節調和システム
自然のリズムと個人の月相パターンが調和するよう、季節的要素を統合

### 3. 成長ストーリー設計
一年を通じて個人が成長していく物語として運勢を構成

### 4. バランス調整機能
特定の時期に偏重しないよう、年間全体でバランスの取れた内容を提供

class AnnualFortuneGenerator {
    generateAnnualFortune(moonPattern, personalityTraits) {
        return {
            annual_theme: this.defineAnnualTheme(moonPattern),
            seasonal_flow: this.generateSeasonalFlow(moonPattern, personalityTraits),
            turning_points: this.identifyTurningPoints(moonPattern),

    }

## 品質保証指標

1. **一貫性**: 年間を通じてテーマが一貫している
2. **自然さ**: 季節の流れと調和している
3. **個別性**: 個人の特性が適切に反映されている
4. **成長性**: 一年を通じた成長ストーリーが描かれている
5. **実用性**: 具体的で実践可能なアドバイスが含まれている

このシステムにより、従来の日々変動する運勢から脱却し、人生の大きな流れを理解できる、より説得力のある年間運勢診断が実現されます。
