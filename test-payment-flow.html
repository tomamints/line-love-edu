決済フローテスト

1. 診断結果ページ

テスト用診断IDでページを開きます：

診断結果ページを開く

期待される動作:

  • プレビューモードで診断結果が表示される
  • 「診断結果を購入する」ボタンが表示される
  • ボタンクリックで決済選択ページへ遷移

2. 決済選択ページ

決済方法選択ページを直接開きます：

決済選択ページを開く

期待される動作:

  • Apple Pay（Safariの場合のみ表示）
  • クレジットカード決済オプション
  • クレジットカード選択でPAY.JP Checkoutモーダルが開く

3. PAY.JP テストカード情報

成功するテストカード:

  • カード番号: 4242 4242 4242 4242
  • 有効期限: 12/30（任意の未来日付）
  • CVC: 123（任意の3桁）
  • 名前: TEST USER

失敗するテストカード:

  • 4000 0000 0000 0002 - カードが拒否される
  • 4000 0000 0000 9995 - 処理エラー

4. 決済フローテスト手順

  1. 診断結果ページを開く
  2. 「診断結果を購入する」ボタンをクリック
  3. 決済選択ページで「クレジットカード」を選択
  4. PAY.JP Checkoutモーダルが開く
  5. テストカード情報を入力
  6. 「支払う」ボタンをクリック
  7. 決済成功後、自動的に完全版ページへリダイレクト（paid=true付き）

5. 決済完了後の確認

決済成功後のページ（テスト用）：

完全版ページを開く（paid=true）

確認ポイント:

  • 完全版のコンテンツが表示される
  • 購入ボタンが非表示になる
  • URLにpaid=trueパラメータが含まれる
