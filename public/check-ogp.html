✅ チェックリスト

  • URLはHTTPS？ → Yes (Vercel自動対応)
  • 画像は1200×630px？ → 確認必要
  • 画像サイズは5MB以下？ → 確認必要
  • og:title設定済み？ → Yes
  • og:description設定済み？ → Yes
  • og:image設定済み？ → Yes
  • og:url設定済み？ → Yes

🚨 トラブルシューティング

画像が表示されない場合：
  1. ブラウザのシークレットモードで確認
  2. 画像URLに直接アクセスして表示確認
  3. Facebook Debuggerで「Scrape Again」をクリック
  4. 24時間待つ（キャッシュの自動更新）
更新が反映されない場合：
  1. URLに?v=2などのパラメータを追加
  2. 各SNSのデバッグツールでキャッシュクリア
  3. og:image:secure_urlタグを追加（HTTPS環境）
